####IP地址

```
reg = /((2[0-4]\d|25[0-5]|[01]?\d\d?)\.){3}(2[0-4]\d|25[0-5]|[01]?\d\d?)/g
```

####座机电话：

```
0\d{2}-\d{8}|0\d{3}-\d{7}
```

####移动电话

```
(139|131|138|188|159)\d{8}
```

####邮箱

```
reg = /^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$/g
```

####QQ号

```
[1-9][0-9]{4,}
```
